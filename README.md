## How to Run Locally
1. Clone this repository:
   ```bash
   git clone https://github.com/eniola-hannah/eniola-hannah.github.io.git
   ```
2. Navigate into the project folder
   ```bash
   cd eniola-hannah.github.io
   ```
3. Open `index.html` in your browser.
